# 데일리 체크 도우미

모바일 RPG 유저를 위한 로그인 없는 데일리 체크리스트 웹앱

## 기능

- 일일/주간 콘텐츠 체크리스트
- 리셋까지 남은 시간 카운트다운 (오전 6시 KST 기준)
- 진행률 표시
- 항목별 ON/OFF 커스텀
- localStorage 저장 (서버/로그인 없음)

## 실행

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인

## 배포

```bash
npm run build
```

Vercel 추천 (GitHub 연동 후 자동 배포)
